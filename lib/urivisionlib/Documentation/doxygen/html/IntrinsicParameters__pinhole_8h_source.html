25  /** Implements the IntrinsicParameters_pinhole class.
26  *
27  * A IntrinsicParameters_pinhole object stores the intrinsic parameters
28  * of an ideal pinhole camera.
29  * A pinhole camera's projection center is at the center of the image and has
30  * no radial distortion. However, the pinhole can have vertical/horizontal ratio
31  * distortion by setting different values for the focal length (in pixels) along
32  * the x and y axes
33  *
34  * Note that I only allow the setting of values for the horizontal and vertical
35  * focal length at the creation of a IntrinsicParameters_pinhole object. There
36  * is no <code>set</code> function for these parameters because this would make
37  * it possible to screw up a <code>IntrinsicParameters_TsaiW</code> object.
38  *
39  * Note that unless dimensions of pixels have been defined
40  * (<code>IntrinsicParameters_CCD</code> and its subclasses), pixel and "metric"
41  * units are identical.
42  *
43  * @author jean-yves herve', 3D Group,
44  * Department of Computer Science and Statistics,
45  * University of Rhode Island
46  *
47  * @see IntrinsicParameters_TsaiW
48  */
