Detailed Description

ImageLaplacian_Sobel class.

There are (so far) 3 possible implementations of the ImageLaplacian_Sobel class. I know that by calling them so I abuse the name "Sobel" but I just call a "Sobel" operator any linear operator that corresponds to a 3x3 covariance mask.
