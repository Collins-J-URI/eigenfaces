ImageLaplacian_Gaussian_I::ImageLaplacian_Gaussian_I ( int  theWidth)

Defines a Laplacian Filter with a specified "radius".

The scale of the filter is the square root of the variance of the associated distrinution. It it also related to the width of the filter by the relation width = ceil(5*scale).

The actual width of the filter = 2*radius + 1

it is set to the smallest odd number greater or equal to the parameter received.

Parameters
theWidthdesired width of the Laplacian filter
