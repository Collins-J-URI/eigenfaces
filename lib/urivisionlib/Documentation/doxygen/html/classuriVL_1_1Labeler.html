Detailed Description

A Labeler object assigns a label to each pixel of its input image.

The main difference between a Labeler and an ImageOperator (and the reason why the former is not a subclass of the latter) is that a Labeler's standard output is a LabelRaster object rather than a RasterImage. Labeler classes provide support for RasterImage objects as output, but really only as a help for development/debugging.
