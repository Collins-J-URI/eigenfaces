Detailed Description

The base virtual class for 3D objects.

A 3D object is characterized by its pose (position and orientation) relative to a given external reference frame and contains its own internal reference frame as well. This means that 3D entities suchs as points and segments are not 3D objects (no complete internal reference frame).
