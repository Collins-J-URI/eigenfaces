44  /** A Pattern2D is defined as a connected set of pixels.
45  * A Pattern2D can be stored either as a <code>Blob</code> or as a rectangular region.
46  * In either case, image information is stored for each pixel of the Pattern.
47  * The indices of the pixels of the blob and of the image stored have been shifted so
48  * that the pattern's upper-left corner is at (0, 0).
