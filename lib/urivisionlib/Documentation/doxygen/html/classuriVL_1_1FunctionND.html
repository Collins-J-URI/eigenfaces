Detailed Description

Pure virtual parent class for all n-dimensional functions.

The function's actual implementation will be done in the child class and it will always be hard-coded.

Note that, contrary to a mathematical function, a Function is always defined on an interval that could be -∞ to +∞, but cannot have "holes" in it.
