Detailed Description

Base rgb color image class.

RasterImage_RGB is the base class for all gray-level image types. This class is built around an 8-bit gray-level QuickTime GWorld buffer, which is itself wrapped into a GraphWorld object.

RasterImage_RGB implements or overrides a number of methods of the base RasterImageClass, and in particular the convertTo<Gray,RGB,RGBa> methods.

Note to class developers: Do not subclass RasterImage_RGB but rather RasterImage_RGBa. Support for RasterImage_RGB is now minimal. Images of this type should not be relied upon for more than reading and displaying. The reason for this is that QuickTime does not officially support 24-bits buffers. So, some function work todays but may break down with a future update of QT. Right now fixing the implementation of 24-bits images is a very low priority for me.
