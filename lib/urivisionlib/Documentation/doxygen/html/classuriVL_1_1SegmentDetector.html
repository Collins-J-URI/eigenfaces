Detailed Description

This provides fast detection of horizontal or vertical line segment in the image.

The input image should have been preprocessed using an edge detection technique and should be a binary image. For fast processing, a block local approch for line detection will be applied.

Author
Jingjing Hu, 3D Group, Department of Computer Science and Statistics, University of Rhode Island
