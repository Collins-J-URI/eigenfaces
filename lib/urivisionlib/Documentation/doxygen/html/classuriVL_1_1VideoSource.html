Detailed Description

VideoSource pure virtual parent class.

All sources of images (live CCTV camera, FireWire camera, analog frame grabber, movie file, image file, folder of image files or image stack) are implemented as subclasses of this class, and provide therefore the same basic interface.

void VideoSource::updateImageRasterStatus_ ( RasterImage *  img)
staticprotected

This function is a bit of a hack.

One problem with subclasses of RasterImage that have redundant storage (e.g. RasterImage_HSV) is that we must make sure that we keep track of which raster stores the reference. This function should be called by any subclass of VideoSource. It ensures that the raster that got read into is marked as reference and modified.

Parameters
imgthe RasterImage object whose status needs updating
