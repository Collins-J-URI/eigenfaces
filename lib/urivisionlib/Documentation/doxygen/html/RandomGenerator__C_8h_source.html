27  /** Pseudorandom number generator wrapper class for the awful standard C function.
28  * <p>Needless to say, nobody should be using that one, I just coded it for debugging
29  * purposes (and to have a lower bound, quality-wise, to compare the other
30  * generators against).</p>
31  * <p>There can only be one object of this class, therefore the class stores that object
32  * as a priveate member and only allows static calls to the generation functions.</p>
