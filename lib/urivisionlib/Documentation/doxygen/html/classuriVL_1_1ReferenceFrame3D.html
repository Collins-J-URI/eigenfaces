Detailed Description

A ReferenceFrame3D is the object relative to which any Entity3D is defined.

The notion of ReferenceFrame3D is closely related to that of RigidTransformation3D. Any ReferenceFrame3D is defined relative to another ReferenceFrame3D, down to the "world" ReferenceFrame3D, which is a static variable of this class. A ReferenceFrame3D is related to its reference ReferenceFrame3D through a RigidTransformation3D object.
