Detailed Description

A Function Class which searches for a closed contour in a given image by applying the "balloon" active contour algorithm of L.

Cohen. The balloon's vertices are connected by linear splines.

Author
Clifford Manzanillo, Department of Computer Science and Statistics, URI
Chris Allen, 3DG, URI [major revision 2009]
jyh, 3DG, URI [major uriVL overhaul 2013]
