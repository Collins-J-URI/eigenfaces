Detailed Description

Implements the Histogram class.

This class computes, stores, and returns a 1D array of long storing the histogram of a given raster image. It can also render this histogram in a window. You should use this class if you want to display an image's histogram (for verification purposes) or want to do your own calculations with the histogram array. Otherwise, the HistogramCalculator class provides a number of functions. such as median computation and the generation of gray thresholding regions

Version history

  • 2007/03/24 [jyh] complete rewrite, combined with ImageHistogram subclass
  • 2005/04/24 [jyh] first implementation.
