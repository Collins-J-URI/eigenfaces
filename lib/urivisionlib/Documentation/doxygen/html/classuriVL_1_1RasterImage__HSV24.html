uriVL::RasterImage_HSV24 Class Reference

Detailed Description

A RasterImage_HSV24 is a RasterImage_RGBa that also stores the hue, saturation, and value for each pixel using only an unsigned char to store the hue, which forces us to scale it from [0, 359] to [0, 255].
