LinearSolver_Cholesky::LinearSolver_Cholesky ( Matrix *  a,
bool  preserveMatrix = false 
)

Constructor.

A LinearSolver_Cholesky is created for a specific matrix. That matrix must be square, positive definite, symmetric, otherwise the call will fail (throw an exception or make an assert call depending on the application settings). Only create a LinearSolver if you are going to reuse the same matrix for different right-side terms, or at least matrices with the same dimensions. Otherwise, just use the static solve function

Parameters
Athe matrix of SLEs to solve
preserveMatrixwhen true, the LinearSolver will keep and work on a copy of the matrix (non-destructive mode).

void LinearSolver_Cholesky::setMatrix ( Matrix *  a,
bool  preserveMatrix = false 
)
pure virtual

Assigns a new matrix to this LinearSolver.

This call only makes sense if the new matrix has the same dimensions as the previous one (it then saves a destruction and an allocation of a matrix). Otherwise, just use the static solve function.

Important: If you don't set preserveMatrix true, then the LinearSolver only stores a reference to matrix it receives as a parameter. It will do calculations in this matrix.

  1. You cannot delete it before you call the solve function,
  2. The matrix's elements will be modified (destructive behavior)
Parameters
athe matrix of SLEs to solve
preserveMatrixwhen true, the LinearSolver will keep and work on a copy of the matrix (non-destructive mode).
