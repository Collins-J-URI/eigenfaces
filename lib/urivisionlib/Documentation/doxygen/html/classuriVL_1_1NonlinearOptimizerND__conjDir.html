Detailed Description

Class that implements the conjugate directions line search optomization algorithm.

The class implements different algorithms for choosing and updating the search directions. At some point in the future, the library may add support for sme sort of dynamic addition of a new update scheme, but for the time being the only way to add an update scheme is to add it statically in this class.
