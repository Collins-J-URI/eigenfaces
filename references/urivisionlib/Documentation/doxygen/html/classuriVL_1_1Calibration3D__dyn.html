void Calibration3D_dyn::calibrate ( const vector< CalibrationPair * > *  calibData,
Camera *  cam 
)

Calibrates a camera using a list of pairs (3D reference point, 2D image point).

Different registration algorithms and types of cameras may require different number of pairs. Check the individual calibration subclasses' documentation. If the camera has initial values for its intrinsic parameters (e.g. focal length) then these may be used as initial values in the computations.

Parameters
calibDatalist of pairs (3D reference point, image point)
camthe camera object to calibrate
