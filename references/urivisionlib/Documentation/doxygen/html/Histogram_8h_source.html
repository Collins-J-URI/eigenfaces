68  /** Implements the Histogram class.
69  *
70  * This class computes, stores, and returns a 1D array of <code>long</code>
71  * storing the histogram of a given raster image. It can also render this
72  * histogram in a window. You should use this class if you want to display
73  * an image's histogram (for verification purposes) or want to do your own
74  * calculations with the histogram array. Otherwise, the
75  * <code>HistogramCalculator</code> class provides a number of functions.
76  * such as median computation and the generation of gray thresholding regions
77  *
78  * Version history
79  * - 2007/03/24 [jyh] complete rewrite, combined with ImageHistogram subclass
80  * - 2005/04/24 [jyh] first implementation.
