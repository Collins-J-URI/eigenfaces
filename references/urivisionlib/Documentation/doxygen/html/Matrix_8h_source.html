47  /**
48  * Implements the Matrix class of Linear Algebra matrices.
49  * <b>Note:</b> Being somewhat inconsistant, I added flags to store information about the
50  * matrix such as whether it is symmetric, orthogonal, upper-triangular, etc. On the one
51  * hand, it might be nice to keep track of that because it allows to be "smarter" when we
52  * need to solve an SLE (e.g. it's good to know if the matrix is orthogonal or triangular)
53  * or compute the determinant (e.g. if the matrix is triangular). On the other hand, it
54  * is a bit of a hassle to keep track of this, and this could be claimed to be the user's
55  * job to do so. At this point, the flags are declared but not accessible, and their
56  * computation/update is not fully implemented.
57  *
58  * @author Joshua Sawyer and jean-yves herve',
59  * 3D Group for Interactive Visualization,
