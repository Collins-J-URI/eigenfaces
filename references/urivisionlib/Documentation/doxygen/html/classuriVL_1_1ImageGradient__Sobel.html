Detailed Description

Vector operator that computes the gradient of a raster image using Sobel-like masks.

Objects of this class only operate on gray-level images. If a color image is sent as parameter, it gets converted to a gray-level image first. For operation on color images, see ColorImageGradient subclasses. 

@author jean-yves herve', 3D Group, 
                        Department of Computer Science and Statistics,
                        University of Rhode Island
