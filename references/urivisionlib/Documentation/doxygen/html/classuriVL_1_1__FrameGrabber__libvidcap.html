Detailed Description

"Generic" implementation of the reading of video data from a live video source.

This class only contains private functions and should therefore never be instantiated directly in user code. Instead, instantiate a FrameGrabber object, which will create (and store a reference to) a _FrameGrabber_libvidcap object.
