Detailed Description

A GrayThresholdLabeler segments an image into regions based on a set of predefined gray-level modes.

Note: So far this class does not do much in the way of data validation. For example, if the user defines gray level or color domains I simply assume that these domains do not overlap, without checking for it.

Author
Fang Han and jean-yves herve', 3D Group, Department of Computer Science and Statistics, University of Rhode Island
