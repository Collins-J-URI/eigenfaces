RasterImage * ImageTransformation::rotate ( const RasterImage *  imgIn,
float  angle,
const ImagePoint *  center = NULL 
)
static

Produces a new RasterImage object that is obtained by rotating the input image by the angle specified and around the centerspecified.

Note that:

  • If no center is pecified, or if a NULL pointer is passed for the center, then the rotatio is performed around the geometric center of the image's valid data rectangle.
  • Rotation angles are measured counterclockwise in degree.
  • Dimensions of the rotated image are converted to nearest ≤ integer.

Note that for For 1-bit images (such as RasterImage_binary objects), the only admissible values for the rotation angle are multiple of 90° (e.g. –90, 90, 180, 270). A NULL value is returned for other angles.

Parameters
imgInthe image to apply the rotation to
anglecounterclockwise rotation angle in degree
centerpointer to the point around which the rotation will be perfomed
Returns
pointer to the rotated image produced;

float  sx,
float  sy,
const ImagePoint *  origin = NULL

static

Produces a new RasterImage object that is obtained by scaling the input image by the amounts specified and from the origin specified.

Note that:

  • If no origin is pecified, or if a NULL pointer is passed for the origin, then the scaling is done from (0, 0).
  • Negative scaling along a direction will result in flipping the image along that direction.
  • Dimensions of the scaled image are converted to nearest ≤ integer, so that any image produced by this function will be at least 1 x 1.
Parameters
imgInthe image to apply the scaling to
sxamount of scaling along the horizontal axis
syamount of scaling along the vertical axis
originpointer to the point from which the scaling will be perfomed
Returns
pointer to the scaled image produced;
