Detailed Description

Class defining a 1D section of an n-dimensional function.

A 1D section of an nD function f is defined as the function

f1: s –> f(x0 + s.v),

where x0 is an n-dimensional point in f's domain, v is an n-dimensional vector/direction, and s is a real number.
