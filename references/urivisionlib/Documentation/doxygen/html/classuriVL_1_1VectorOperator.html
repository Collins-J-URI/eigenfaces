virtual void uriVL::VectorOperator::applyInto ( const RasterImage *  imgIn,
VectorField *  vectOut 
)
pure virtual

Applies this operator to a RasterImage and writes the result into a VectorField received as parameter.

This function will be overloaded by subclasses of VectorOperator. These subclasses should follow the following rules:

  • if the operator is an "integer" operator, then the RasterImage returned should have the same type as the input image
  • if the operator is a "float" or "double" operator, then the RasterImage returned should be of the "float" form of the input image. For example, if imgIn was RasterImage_Gray, then the image returned should be RasterImage_Gray_F.
Parameters
imgInthe RasterImage to which the operator is applied
theRectthe rectangle within which to apply the filter
vectOutthe VectorField into which the output is written
