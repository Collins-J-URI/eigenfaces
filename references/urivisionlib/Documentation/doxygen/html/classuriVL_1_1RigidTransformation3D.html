Detailed Description

A RigidTransformation is a transformation between reference frames that preserves angles and lengths.

A RigidTransformation is composed of a rotation and a translation and can be represented either by an object of each type or by a 4x4 homogeneous transformation matrix.

Author
jean-yves herve', 3D Group, Department of Computer Science and Statistics, University of Rhode Island
