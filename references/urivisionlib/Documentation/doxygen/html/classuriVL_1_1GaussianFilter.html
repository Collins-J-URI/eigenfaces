Detailed Description

Pure Virtual parent class for the Gaussian filter classes _F and _I.

Note that in the current implementation of this class the same scale is applied along the x and y directions. Implementing a more general form is a low-priority item at this point.

Author
jean-yves herve', 3D Group, Department of Computer Science and Statistics, University of Rhode Island
