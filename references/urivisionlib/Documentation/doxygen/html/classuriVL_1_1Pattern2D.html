Detailed Description

A Pattern2D is defined as a connected set of pixels.

A Pattern2D can be stored either as a Blob or as a rectangular region. In either case, image information is stored for each pixel of the Pattern. The indices of the pixels of the blob and of the image stored have been shifted so that the pattern's upper-left corner is at (0, 0).
