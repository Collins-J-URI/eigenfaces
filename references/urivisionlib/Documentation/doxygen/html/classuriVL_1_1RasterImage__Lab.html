float ** RasterImage_Lab::getRaster ( void  )
virtual

Returns the base raster as a read-only 1D array.

This method is implemented by the base subclasses of RasterImage, and should be overridden by these classes' own subclasses. If the base raster has not already been allocated, then it should be allocated and initialized using the reference information for the image (for example, HSV data should be converted into RGBA data in the 1D base raster).

Returns
1D base raster for this image
