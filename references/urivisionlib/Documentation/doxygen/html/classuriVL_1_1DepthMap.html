Detailed Description

depth map class A depth map is a form of raster image that instead of storing an image itensity at each image point stores a depth value.

A DepthMap object can be rendered as a raster image object or drawn as a 3D mesh.

A DepthMap object stores an elevation map:

  • for a dense depth map, as a 2D raster Z[Y, X] (the row index of a raster corresponds to the Y coordinate, the column corresponds to X);
  • for a sparse depth map, it is assumed that the 3D points form an elevation surface along the Z direction

The depth map (for me at least) is pretty much indisossiable from the notion of perspective projection. Therefore, all constructors for this class require an IntrinsicParameters_pinhole object (which will provide a focal length and scaling parameters for the inverse projection. If the camera used in orthographic, then the raster does not really store a depth map but an elevation map.

So far the only way to create a DepthMap, RangeMap, or ElevationMap object is to send it either a raster or a list of points to initialize it with. In other words, there is no "create now and allocate/initialize later" constructor.
