Detailed Description

Implements the ImagePoint class.

The ImagePoint class is based on the notion of a pixel as a dimensionless point. By opposition, QuickTime sees a pixel as a little square that hangs between the points (xLeft, yTop) and (xLeft+1, yTop+1).

Version history

  • 2004/04/27 [jyh] fixed the pixel-as-point vs. pixel-as-rectangle issue.
  • 2004/02/20 [jyh] first implementation.
